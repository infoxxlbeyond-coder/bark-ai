# BARK AI

BARK AI is a live Web3 AI utility on BNB Chain focused on lost pet matching and private wallet-based contact.

## Summary
BARK AI allows users to upload a lost pet photo, search for matches with AI, and contact the owner privately through wallet-based messaging.
